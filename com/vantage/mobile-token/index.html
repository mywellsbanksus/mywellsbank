Mobile token

A fast and convenient way to authenticate

With mobile token, you can use your iOS or AndroidTM device to access your Wells Fargo VantageSM services and perform sensitive transactions from your desktop, laptop, tablet, or mobile device.

You no longer need to worry about missing a deadline because you forgot or misplaced your token. As long as you have your mobile device with you, you will always have your token, too.

Mobile token uses proven RSA SecurID® technology that lets you generate a one-time code on your mobile device that you can use to sign on and access secure online banking services. Mobile token provides an added level of security to help protect your accounts and information, and there’s no additional charge to use the feature.

How to get started with mobile token

If you don't already have the Wells Fargo VantageSM app on your iOS or Android device, you'll need to download it from the Apple App Store® or Google PlayTM store.

To set up the mobile token feature:

  1. Sign on to the app on your mobile device. Navigate to your User Profile then select App Settings.
  2. Switch the Mobile token toggle on to enable.
  3. Follow the installation instructions.

Completing transactions using mobile token

With desktop and laptop devices — Open the mobile app to view the token code whenever you need a passcode. Then enter your PIN and the code that appears on your mobile device in the passcode field.

With the mobile app — Enter only your mobile token PIN when prompted. You don't need to enter a token code as it is automatically generated and prefilled for your convenience.

Frequently asked questions

About mobile token

About mobile token

What is mobile token?

The mobile token feature allows you to authenticate using your mobile device in place of your RSA SecurID® token. Mobile token generates a single-use security code on your mobile device that you can use to access your secure Wells Fargo VantageSM services.

Can I use mobile token to access Vantage from my desktop or laptop?

Yes, you can use the mobile token to access your secure services on your laptop or desktop computer. Simply use the code on your mobile device in place of your RSA SecurID token code.

Can I continue using my RSA SecurID token?

Yes, you can use your RSA SecurID token any time you choose.

Who can use mobile token?

You must be entitled to use one or more products requiring token authentication to use mobile token.

Mobile token is available to eligible Wells Fargo Vantage℠ iOS and Android™ app users. For the best experience, you should download the latest version of the app available for your device from the Apple App Store or Google Play™ store.

How much does it cost to use the feature?

There's no additional cost to use the mobile token. You'll need to download the mobile app from the Apple App Store or Google Play store to use mobile token.

What are the potential benefits of using mobile token?

The mobile token is a fast and convenient way to access your secure banking services. With the mobile token, you no longer have to worry about missing a deadline because you forgot or misplaced your token. You can quickly access your banking services with something you usually have available — your mobile device.

Why would I use mobile token?

Mobile token is available immediately after setting it up in the app making it a convenient option for accessing your secure banking services. You can generate a token on your mobile device any time you don't have your physical security token on hand.

Security

Security

Is mobile token as secure as an RSA SecurID token?

Mobile Token uses proven RSA SecurID technology to generate your token code. The feature delivers a security code to your mobile device rather than a physical security token.

Using mobile token

Using mobile token

How do I get started with mobile token?

First, you'll need to download the Wells Fargo Vantage℠ app from the Apple App Store or Google Play store. Then, follow these steps:

  • Sign on to the app on your mobile device. Navigate to your User Profile then select App Settings.
  • Switch the Mobile token toggle on to enable. 
  • Follow the installation instructions.

For more information, visit the Mobile Overview User Training.

Do I need to enter my credentials every time I enter my token code?

When accessing Vantage from your laptop or desktop, you'll need to enter your PIN and token code, yet when using the mobile app, you'll only need to enter your PIN.

I have more than one company ID or user ID. Can I use my mobile token for all of them?

Mobile token can only be used with one company and user ID per mobile device.

Which services can I access with mobile token?

You can use mobile token for any service that requires you to enter a RSA SecurID token code.

What if I lose or replace my mobile device?

The mobile token can only be installed on one device at a time, so if you get a new mobile device in the future or you just want to use one of your other mobile devices instead, then you'll go through the setup process to enable mobile token on your new device. You will be able to use your existing PIN on the new device.
