Preguntas principales

¿Cómo descargo e instalo un nuevo navegador?

Antes de descargar un nuevo navegador, asegúrese de que su sistema operativo y el nuevo navegador cumplan con los requisitos técnicos de Wells Fargo. También ofrecemos instrucciones simples sobre cómo descargar e instalar navegadores compatibles.

¿Cómo activo JavaScript en mi navegador?

Para su seguridad, debe tener JavaScript activado en su navegador para iniciar sesión en la Banca por Internet Wells Fargo Online®.

Java se puede activar en las siguientes plataformas: Windows 10, Windows 8, Windows 7, Windows Vista, Windows XP y Mac OS X.

Chrome: si tiene problemas para acceder a las aplicaciones Java usando Chrome, Oracle recomienda utilizar Microsoft Edge (Windows) o Safari (Mac OS X) en su lugar.

¿Por qué un enlace no funciona?

Si hace clic en un enlace que no funciona, puede deberse a que se están bloqueando las ventanas emergentes a las que solicitó acceder. Para abrir el enlace, puede desactivar el bloqueador de ventanas emergentes o mantener presionada la tecla CTRL mientras hace clic en el enlace. Es posible que los usuarios de Windows 10 que usen Internet Explorer vean una barra de información debajo de la barra de direcciones en el navegador donde se les indica que se ha bloqueado una ventana emergente. Haga clic en esta barra de información y elija Temporarily allow pop-ups (Permitir elementos emergentes temporalmente), y luego vuelva a hacer clic en el enlace para abrir la ventana emergente.

También puede elegir:

  • Always Allow Pop-Ups from this Site (Permitir siempre elementos emergentes de este sitio).
  • Ingrese *.wellsfargo.com/spanish en la casilla "Address of website to allow" (Dirección del sitio web que desea permitir). (No olvide incluir el *).
  • Haga clic en Add (Agregar).

Compatibilidad con navegadores

¿Qué ajustes de navegador necesito para utilizar la banca por Internet Wells Fargo Online®?

Para poder iniciar sesión en la banca por Internet Wells Fargo Online, su navegador debe tener habilitados JavaScript y cookies.

¿Por qué es necesario que actualice mi navegador?

Por motivos de seguridad, dado que, ocasionalmente, "vence" la habilidad de los navegadores para llevar a cabo transacciones confidenciales, como operaciones bancarias por Internet. A fin de mantener nuestro alto nivel de seguridad para los clientes, Wells Fargo no le permite acceder a sus cuentas a través de la Banca por Internet Wells Fargo Online con un navegador que haya vencido. Para continuar accediendo a su(s) cuenta(s) de Wells Fargo por Internet, algunos clientes deberán actualizar la versión de su navegador preferido a una versión compatible más reciente. Consulte nuestra lista de navegadores compatibles.

¿Qué hago si mi nuevo navegador aún no es compatible?

Si está utilizando la última versión de su navegador, tenga en cuenta que puede haber una demora hasta que sea compatible la versión del nuevo navegador mientras la revisamos para asegurarnos de que cumpla nuestros estándares de seguridad. Le recomendamos que descargue e instale un navegador compatible hasta que la nueva versión compatible se encuentre disponible oficialmente.

¿Cómo puedo ajustar el navegador para que acepte cookies?

Las cookies son fragmentos de información que se entregan al navegador de su computadora. Estos datos se devuelven al servidor cada vez que accede a una nueva página.

El objetivo de activar las cookies es ayudarles a los sitios Web a identificar usuarios que regresan al sitio Web para proporcionales páginas personalizadas. No podrá acceder a ninguna página segura si elige Block All Cookies (Bloquear todas las cookies) en los ajustes de privacidad. Siga los pasos a continuación que correspondan para activar las cookies en su computadora:

Microsoft Edge (en inglés)

Safari para Macintosh OS X (en inglés)

Firefox para Windows (en inglés)

Google Chrome (en inglés)

¿Por qué la respuesta que estoy recibiendo es excepcionalmente lenta o aparece un mensaje indicando que el navegador no puede abrir la página Web?

Son muchas las causas posibles para este mensaje de error o tiempos de respuesta lentos. Las dos causas más comunes son: wellsfargo.com/spanish aparece como un sitio restringido en su navegador, o bien, es preciso borrar la caché o los archivos temporales usados por el navegador.

Compatibilidad con ventanas emergentes

¿Qué es una ventana emergente?

Las ventanas emergentes son pequeñas ventanas del navegador Web que contienen información, o pueden ser anuncios que aparecen por encima o por debajo de la página Web que esté viendo en ese momento. Por lo general, se usan para publicidad y aparecen en forma automática independientemente de que usted desee verlas o no. Wells Fargo no las utiliza para publicidad. Pero también es posible que aparezcan si usted así lo solicita para mostrarle texto de ayuda u otra página de un sitio que sea informativa.

De vez en cuando, es posible que una nueva ventana del navegador se abra cuando haga clic en un enlace de nuestro sitio. Estas ventanas emergentes iniciadas por el usuario solo se abren después de que usted las solicite haciendo clic en un enlace. Si hace clic en un enlace que no funciona, puede deberse a que en su equipo se están bloqueando las ventanas emergentes a las que solicitó acceder.

¿Cómo funcionan los bloqueadores de ventanas emergentes?

Los bloqueadores de ventanas emergentes evitan que se abran ventanas emergentes según los ajustes que haya seleccionado para el bloqueador.
