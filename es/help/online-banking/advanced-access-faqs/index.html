Preguntas generales

¿Qué es el acceso avanzado?

El acceso avanzado es un servicio que le ofrece un nivel de seguridad adicional para proteger mejor su información y ayudar a evitar transacciones no autorizadas.

¿Cuándo necesitaré el acceso avanzado?

A fin de proteger su información personal y proteger sus cuentas contra el acceso no autorizado, podríamos requerir un código cuando:

  • Acceda a ciertas cuentas o servicios de Wells Fargo.
  • Envíe dinero a otra persona a quien no haya transferido dinero antes, o reciba dinero de una cuenta que no sea de Wells Fargo que usted no haya usado antes.

¿Cómo me protege el acceso avanzado?

El acceso avanzado es otro medio que nos permite verificar su identidad. Al pedirle que reciba un código en su teléfono, ayudamos a evitar que alguien que no sea usted acceda a su información sensible.

¿Qué sucede si necesito un código de acceso avanzado pero no tengo acceso a los dispositivos que uso normalmente para recibir el código?

Llámenos al 1-866-609-3037 (en inglés) para pedir ayuda. Si llama desde fuera de Estados Unidos, use uno de nuestros códigos de acceso internacional gratuitos.

¿Cómo puedo obtener un código de acceso avanzado si estoy de viaje fuera de EE. UU.?

Podría tener la opción de recibir su código de acceso avanzado mediante una notificación push en su app de Wells Fargo Mobile® si está de viaje en el extranjero. También puede llamarnos al número gratuito desde la mayoría de los países fuera de EE. UU. Verificaremos su identidad y le proporcionaremos la ayuda que necesita.

¿Qué puedo hacer si se me bloquea el servicio de acceso avanzado?

Llámenos al 1-866-609-3037 (en inglés) para pedir ayuda.

¿Cómo sabré si la llamada telefónica con mi código de acceso avanzado realmente proviene de Wells Fargo?

Las llamadas que reciba con un código de acceso avanzado provienen del número 1-866-609-3037. Le recomendamos que agregue este número a su lista de contactos para que pueda identificar rápidamente nuestras llamadas.

También describiremos lo que usted está haciendo cuando le proporcionemos el código. Por ejemplo, le informaremos que el código es para finalizar la transferencia. De esa forma, podrá confirmar que estamos respondiendo a su solicitud específica.

¿Cómo puedo saber que el mensaje de texto con mi código realmente proviene de Wells Fargo?

Enviamos nuestros mensajes de texto con el código de acceso avanzado desde el 93557 y el 93733. Le recomendamos que agregue estos números a su lista de contactos telefónicos para que pueda identificar fácilmente nuestros mensajes de texto.

RSA SecurID®

¿Puedo utilizar un dispositivo RSA SecurID con el acceso avanzado?

Si tiene un dispositivo RSA SecurID de Wells Fargo, le daremos la opción de utilizar un código proveniente de su dispositivo cuando se requiera el acceso avanzado.

¿Qué es un dispositivo RSA SecurID y cómo funciona?

El dispositivo RSA SecurID es un pequeño dispositivo electrónico portátil que se puede colocar fácilmente en un llavero. Cada 60 segundos, el dispositivo genera y muestra un código de acceso numérico aleatorio único. Si usted tiene un dispositivo RSA SecurID de Wells Fargo, podrá usar códigos generados por el dispositivo cuando se requiera el acceso avanzado.

¿Qué debo hacer para obtener un dispositivo RSA SecurID nuevo o de reemplazo?

Puede solicitar un nuevo dispositivo por un cargo por única vez no reembolsable de $25, que incluye envío y manejo. Inicie sesión en la banca por Internet en WellsFargo.com/es para realizar su pedido. No hay costos ni cargos asociados con el reemplazo de un dispositivo RSA SecurID.

Seguridad en las compras por Internet

¿Qué es Visa Secure?

Visa Secure es un servicio ofrecido por Visa y Wells Fargo para ayudar a proteger a los titulares de tarjeta frente a compras por Internet no autorizadas. El servicio evalúa cada transacción y, en algunos casos, se le pedirá que ingrese un código secreto por única vez para completar la transacción.

¿Qué es American Express SafeKey®?

SafeKey es un servicio ofrecido por American Express y Wells Fargo para ayudar a proteger a los titulares de tarjeta frente a compras por Internet no autorizadas. El servicio evalúa cada transacción y, en algunos casos, se le pedirá que ingrese un código secreto por única vez para completar la transacción.

¿Se requerirá un código secreto por única vez para todas las compras por Internet?

No. Wells Fargo está trabajando para ayudar a mantener su tarjeta segura y solo le solicitará un código secreto cuando se requiera verificación adicional.

¿Cómo recibiré el código secreto?

El código secreto por única vez puede ser enviado por mensaje de texto o notificación push. Para envío por mensaje de texto, mostraremos los números de teléfono móvil que tenemos registrados, y se le pedirá que seleccione el número deseado. Enviamos nuestros códigos secretos por única vez desde el 93557 y el 93733. Le recomendamos que agregue los números a su lista de contactos telefónicos para que pueda identificar nuestros mensajes de texto. Si tiene la app de Wells Fargo Mobile®, puede optar por que el código se envíe como una notificación push a su dispositivo móvil.

¿Qué debo hacer si no solicité el código secreto?

Si recibe un código secreto por única vez que usted no solicitó en relación con una compra por Internet, llame al número que aparece al dorso de su tarjeta para hablar con un representante de Wells Fargo.

¿Qué sucede si no reconozco los números de móvil entre los que puedo elegir para que Wells Fargo envíe el código secreto?

Si no reconoce los números, llame al número que aparece al dorso de su tarjeta para hablar con un representante de Wells Fargo.

Es posible que se requiera inscripción. La disponibilidad podría verse afectada por el área de cobertura de su proveedor de telefonía móvil. Es posible que se apliquen tarifas por servicio de mensajería y datos de su proveedor de telefonía móvil.

RSA SecurID es una marca comercial registrada de RSA Security LLC.
