Acceso avanzado

  • Imprima
Seguridad adicional, cuando más la necesita

El acceso avanzado es un nivel de seguridad adicional que le ayuda a proteger su información y evitar transacciones no autorizadas.

Así es como funciona

Solicite un código

Al realizar operaciones en la Banca por Internet o la Banca Móvil, a veces podríamos indicarle que solicite un código de acceso para realizar determinadas transacciones o acceder a información sensible.

Ingrese el código

Si está disponible, puede recibir su código de acceso mediante un mensaje de correo electrónico, un mensaje de texto SMS, una llamada a su teléfono o una notificación push a su app de Wells Fargo Mobile.

Código aprobado

Verificaremos el código y usted podrá continuar con su transacción o solicitud.

Enviamos nuestros mensajes de texto con el código de acceso avanzado desde el 93557 y el 93733. Le recomendamos que agregue estos números a su lista de contactos telefónicos para que pueda identificar fácilmente nuestros mensajes de texto. Es posible que se apliquen las tarifas por servicio de mensajería y datos de su proveedor de telefonía móvil.
